## Contributing to Bridge

Bridge is an [Apache Software foundation](https://www.apache.org/theapacheway/index.html) project.
The core contributors (or committers) to Bridge communicate primarily in the following channels (
which can be joined by anyone):

- [Mailing list](https://lists.apache.org/list.html?dev@bridge.apache.org)
- [Apache Bridge Slack community](https://join.slack.com/t/apache-bridge/shared_invite/zt-1jp6hjzrq-H0PlFtToyLWuPiJDuRWCNw)
- [GitHub issues and PR's](https://github.com/kiranbmore0101/bridge/issues)

More references:
- [Comprehensive Tutorial for Contributing Code to Apache Bridge](https://preset.io/blog/tutorial-contributing-code-to-apache-bridge/)
- [CONTRIBUTING Guide on GitHub](https://github.com/kiranbmore0101/bridge/blob/master/CONTRIBUTING.md)
